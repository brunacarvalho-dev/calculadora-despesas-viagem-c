# 💰 Calculadora de Despesas de Viagem em C
Projeto desenvolvido em linguagem C com o objetivo de calcular o custo total de combustível em uma viagem, a partir da distância percorrida e do preço do combustível.

📌 Sobre o projeto
Este programa foi criado para aplicar conceitos básicos de lógica de programação utilizando a linguagem C, como:

* Entrada e saída de dados
* Uso de variáveis
* Estruturas de repetição (`while`)
* Estruturas condicionais (`if/else`)
* Validação de dados

O sistema permite que o usuário insira valores válidos e realiza o cálculo automaticamente.

⚙️ Funcionalidades
✔️ Entrada de distância da viagem
✔️ Entrada do preço do combustível
✔️ Validação de valores (não aceita números negativos ou zero)
✔️ Cálculo automático do custo total
✔️ Exibição formatada com duas casas decimais
